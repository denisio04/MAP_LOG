# 📍 MAP LOG

**MAP LOG** is a geographic logbook mobile application designed with a **Brutalist** aesthetic. It allows you to record precise locations on a map, organize them into custom folders, and enrich each entry with detailed notes and photographs.

It is the ideal tool for travelers, urban explorers, or anyone needing a visual and geolocalized record of their points of interest.

## ✨ Key Features

- **🗺️ Interactive Map**: Add markers by long-pressing any point on the map.
- **📂 Folder Organization**: Classify your places by categories (e.g., Restaurants, Work, Travel).
- **📸 Multimedia Attachments**: Upload photos directly to your notes to remember every detail.
- **🕒 Automatic Timestamps**: Precise recording of the creation date and time for every entry.
- **🚗 External GPS Navigation**: Direct button to open routes in Google Maps or Apple Maps.
- **🔄 Data Management**: Export and import your entire database in JSON format (Backups).
- **📏 Location Intelligence**: Real-time distance calculation from your GPS position and proximity-based sorting.
- **🎨 Brutalist Aesthetic**: Raw, direct, and high-contrast interface with native **Dark Mode** support.

## 🛠️ Tech Stack

- **Frontend**: React Native + Expo (SDK 54)
- **State Management**: Zustand (Lightweight and efficient)
- **Persistence**: AsyncStorage
- **Maps**: React Native Maps
- **Styling**: Vanilla CSS-in-JS (Custom theming system)

## 🚀 Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/denisio04/MAP_LOG.git
   cd MAP_LOG
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the project**:
   ```bash
   npx expo start
   ```
